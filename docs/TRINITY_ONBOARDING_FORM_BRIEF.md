# Artificial Trinity Labs AI - Client Onboarding Form Brief

## Purpose
Create a comprehensive onboarding form to collect all necessary information from new clients to set up their AI voice calling campaigns effectively.

## Form Sections & Fields

### 1. Company Information
- **Company Name** (required)
- **Industry/Sector** (dropdown: Real Estate, Insurance, Solar, B2B Services, Healthcare, Other)
- **Website URL**
- **Company Registration Number** (optional)

### 2. Primary Contact Details
- **Main Contact Full Name** (required)
- **Job Title/Position**
- **Email Address** (required)
- **Direct Phone Number** (required)
- **Preferred Communication Method** (Email/Phone/WhatsApp)
- **Best Time to Contact**

### 3. Campaign Team Information
- **Team/Department Name** (e.g., "Sales Team", "Customer Success")
- **Team Size** (number of agents/users)
- **Team Leader Name**
- **Team Leader Email**
- **Additional Team Members** (optional, repeatable fields)

### 4. Campaign Details
- **Campaign Name** (required)
- **Campaign Objective** (dropdown: Lead Generation, Appointment Setting, Customer Follow-up, Survey Collection, Debt Collection, Other)
- **Target Audience Description**
- **Expected Call Volume** (per day/week/month)
- **Campaign Duration** (Start Date - End Date)
- **Geographic Target** (Countries/Regions)

### 5. Product/Service Information
- **Product/Service Name** (required)
- **Product Description** (text area)
- **Key Selling Points** (bullet points, up to 5)
- **Price Range** (if applicable)
- **Common Objections & Responses** (text area)
- **Unique Value Proposition**

### 6. Call Script Requirements
- **Existing Script Available?** (Yes/No)
  - If Yes: Upload script document
  - If No: Request script creation service? (Yes/No)
- **Call Opening Preference** (formal/casual/customized)
- **Key Questions to Ask** (text area)
- **Qualifying Criteria** (what makes a lead qualified?)
- **Call-to-Action** (what action should prospects take?)
- **Objection Handling Guidelines**

### 7. Voice & Tone Preferences
- **AI Voice Gender** (Male/Female/Neutral)
- **Accent Preference** (British/American/Australian/Neutral)
- **Speaking Pace** (Slow/Normal/Fast)
- **Tone of Voice** (multiple select):
  - Professional
  - Friendly/Warm
  - Authoritative
  - Conversational
  - Enthusiastic
  - Gentle/Soft
  - Confident
  - Consultative
  - Urgent
  - Empathetic
- **Personality Traits** (describe ideal agent personality)

### 8. Technical Setup
- **Forwarding Phone Numbers** (where to transfer interested prospects)
  - Primary Transfer Number (required)
  - Backup Transfer Number (optional)
  - Transfer Hours (when transfers are accepted)
- **Caller ID Display Number** (if different from system number)
- **Call Recording Consent** (Yes/No)
- **Preferred CRM System** (Salesforce/HubSpot/Pipedrive/Other/None)
- **Webhook URL** (for real-time data push, optional)

### 9. Working Hours & Scheduling
- **Business Hours** (for each day of the week)
  - Monday: [Start Time] - [End Time] or Closed
  - Tuesday: [Start Time] - [End Time] or Closed
  - Wednesday: [Start Time] - [End Time] or Closed
  - Thursday: [Start Time] - [End Time] or Closed
  - Friday: [Start Time] - [End Time] or Closed
  - Saturday: [Start Time] - [End Time] or Closed
  - Sunday: [Start Time] - [End Time] or Closed
- **Time Zone** (dropdown)
- **Holiday Schedule** (list of dates to exclude)
- **Maximum Calls Per Day** (rate limiting)
- **Preferred Calling Times** (morning/afternoon/evening)

### 10. Data & Lead Management
- **Do you have existing data to use?** (Yes/No)
  - If Yes:
    - Data Format (CSV/Excel/CRM Export/Other)
    - Number of Records
    - Data Quality (Verified/Unverified/Mixed)
    - Upload Data File or Schedule Import
- **Lead Source** (Website/Purchased List/Event/Other)
- **Data Compliance**:
  - GDPR Compliant? (Yes/No)
  - Opt-in Confirmation Available? (Yes/No)
  - DNC List Management Required? (Yes/No)

### 11. Success Metrics & Reporting
- **Primary KPI** (Appointments Set/Leads Qualified/Sales Closed/Other)
- **Target Conversion Rate** (percentage)
- **Reporting Frequency** (Daily/Weekly/Monthly)
- **Report Recipients** (email addresses)
- **Custom Metrics to Track** (optional, text area)

### 12. Compliance & Legal
- **Industry Regulations** (TCPA/GDPR/PECR/Other/None)
- **Consent Recording Required?** (Yes/No)
- **Disclosure Requirements** (any specific disclosures needed)
- **Age Verification Required?** (Yes/No)
- **Geographic Restrictions** (areas to exclude)

### 13. Budget & Billing
- **Expected Monthly Budget** (range)
- **Billing Contact Name**
- **Billing Email**
- **VAT/Tax ID Number**
- **Purchase Order Required?** (Yes/No)
- **Payment Terms Preference** (Monthly/Quarterly/Annual)

### 14. Integration Requirements
- **Current Phone System** (VoIP provider if any)
- **Marketing Automation Platform** (if any)
- **Required Integrations** (checklist):
  - Slack notifications
  - Email notifications
  - SMS follow-ups
  - Calendar booking
  - CRM sync
  - Custom API
  - Zapier/Make

### 15. Additional Information
- **Previous AI/Automation Experience** (None/Some/Extensive)
- **Main Challenges to Solve**
- **Success Criteria** (what would make this successful?)
- **Special Requirements** (text area)
- **Preferred Go-Live Date**
- **Pilot/Testing Phase Required?** (Yes/No)

### 16. Terms & Agreements
- **Terms of Service** (checkbox, required)
- **Privacy Policy** (checkbox, required)
- **Data Processing Agreement** (checkbox, required)
- **Communication Consent** (checkbox for marketing communications)

## Form Features to Include

1. **Progress Indicator** - Show completion percentage
2. **Save & Continue Later** - Allow partial completion
3. **Field Validation** - Real-time validation for emails, phones, URLs
4. **Conditional Logic** - Show/hide fields based on answers
5. **File Upload** - For scripts, data files, etc. (max 10MB)
6. **Multi-step Layout** - Break into logical sections
7. **Mobile Responsive** - Works on all devices
8. **Confirmation Page** - Summary of submitted information
9. **Email Confirmation** - Send copy to client
10. **Integration** - Connect to CRM/database

## Post-Submission Actions

1. Send confirmation email to client
2. Create account in system
3. Notify sales team
4. Schedule onboarding call
5. Generate campaign setup checklist
6. Create project in project management tool
